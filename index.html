    GitHub Depo Bağlantısı (Birincil Depolama)

    📌 Nasıl Çalışır?

    Tüm stok verileriniz GitHub reponuzda bir JSON dosyası olarak saklanır. Her işlemde otomatik olarak GitHub'a commit atılır. Personal Access Token (PAT) gereklidir.

    1. GitHub'da Settings > Developer settings > Personal access tokens > Tokens (classic) kısmına gidin.
    2. Generate new token (classic) tıklayın.
    3. repo (Full control) ve contents yetkilerini işaretleyin.
    4. Oluşturulan token'ı aşağıdaki alana yapıştırın.
    5. Repo olarak mustafaorhan-dev/STOKDOSYA kullanın.
    // app.js — Kalıcı ayarlar (tarayıcı silinse bile durur)
const HARD_CODED_GITHUB = {
  owner: 'mustafaorhan-dev',
  repo: 'STOKDOSYA',
  path: 'data/stok.json',
  token: 'ghp_...' // isteğe bağlı
};

    ⚠ Güvenlik Uyarısı

    Token tarayıcı localStorage'a kaydedilir. Başkalarıyla paylaşmayın. Token'ın sadece bu repo için repo (contents) yetkisi olması önerilir. GitHub Pages üzerinde çalışırken token URL'de görünmez, güvendedir.
